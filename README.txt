Masters thesis on features of the most prominent MMA fighters based on statistics scrapped from official page on UFC fight nights statistics.

Chapters 1 and 2 include statistical methods selected and their description along with some chart used to visualize different parameters and methods.
Chapter 3 includes:
- scrapping of data from the UFC statistics site
- Exploratory Data Analysis/Descriptive statistics using pandas and matplotlib
- Feature Selection and transformation using scikitlearn and pandas
- Statistical modeling using scikitlearn decision tree and logistic regression
- Summary of the most important features based on the analysis
